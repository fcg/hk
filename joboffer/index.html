AIP 加拿大大西洋四省移民试点项目

加拿大大西洋四省试点移民项目 AIPP，Atlantic Immigration Pilot Program 是2017年新推出的3年计划，2017年全年4省总共 2000配额，其中 NS 省独占 800，也就是 40%，AIPP 面向的是在大西洋四省找到工作的当地临时工人。

AIPP 具体又划分为三个小类 - 飞出国：

  • AHSP 大西洋高技能移民项目 Atlantic High-Skilled Program
  • AISP 大西洋中等技能移民项目 Atlantic Intermediate-Skilled Program
  • AIGP 大西洋国际留学毕业生移民项目 Atlantic International Graduate Program
AIP 要求 AISP（半技术工） AHSP（技术工） AIGP（海洋省份留学生）
工作经验 近3年内1年 Level C 近3年内1年 Level 0，A, B 不要求
当地 job offer 全职，永居，LEVEL C及以上 全职，永居获得后1年，LEVEL B及以上 全职，永居获得后还有最少1年期限，Level C及以上
教育 高中及以上学历 高中及以上学历 认可学校最少2年高等或职业教育，在加拿大实际学习2年，毕业一年内，近2年内在大西洋省份生活16个月以上

美国雇主担保移民 - flyabroad

随着美国移民政策的收紧，很多在美多年的留学生也没法留在美国。随着美国经济向好，很多优秀的人也会前往美国。美国是一个移民大国，但也不缺想要移民美国的优秀人才，因此，美国技术移民政策，主要是雇主移民政策会门槛提高。

对条件很好的可以考虑 EB1，EB2，EB3 技术工作，对条件一般申请人只能考虑 EB3 非技术工，当前 EB3 合格雇主越来越少，且排期长，申请雇主担保也需要尽早。
